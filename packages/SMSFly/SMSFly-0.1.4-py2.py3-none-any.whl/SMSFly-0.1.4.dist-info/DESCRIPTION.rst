# python-smsfly
SMS-Fly gateway API Python package

## Development
```sh
pip install -e .[test,dev]
nosetests --ipdb
```

## Testing
```sh
pip install -e .[test]
nosetests
```

## Usage
```python
from smsfly import SMSFlyAPI
api = SMSFlyAPI(account_id='3801234567', account_pass='qwerty')
api.getbalance()
```
