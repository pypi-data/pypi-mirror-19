The Islet Regulome Browser

The Islet Regulome browser was designed by Dr. Lorenzo Pasquali (IGTP/IJC) and Dr. Loris Mularoni (IRB) who implemented and developed both the code for generating plots and the web interface. Initial supervision for the project was provided by Prof. Jorge Ferrer (ICL) at Institut D'Investigacions Biomediques August Pi i Sunyer (IDIBAPS).

The Islet Regulome Browser is now developed and maintained in the Pasquali Lab by Mireia Ramos and Dr. Loris Mularoni with support from the Program for Predictive and Personalized Medicine of Cancer (PMPPC) at the Institute Germans Trias i Pujol (IGTP) Bioinformatics Core.
